# Conversor de unidades

Esse é um projeto pessoal meu para preencher portfólio, se baseia em converter unidades de temperatura e de distância

Essas medidas são: Celsius, Fahrenheit e Kelvin para temperatura e o sistema métrico e imperial para distâncias.
